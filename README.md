JS is ...

- High-Level programming language
- Interpreted or just-in-time compilation
- Prototype-based object-oriented
- Multi-paradigm (Procedural, OOP, Functional programming )
- First-class function
- Dynamic
- Single-threaded
- Non-blocking event loop
- Garbage-collected
